# 송폼 구조

## 기본 구조
```
[Verse 1] → [Verse 2] → [Chorus] → [Bridge] → [Final Chorus] → [Outro]
```

## 각 섹션 역할
| 섹션 | 역할 | 줄 수 |
|------|------|-------|
| Verse 1 | 눈에 보이는 장면 (카메라) | 4줄 (AAB는 3줄) |
| Verse 2 | 내면으로 진입 (독백) | 4줄 (AAB는 3줄) |
| Chorus | 감정 폭발 + 훅 (제목 반복) | 4줄 (AAB는 3줄) |
| Bridge | 반전 or 고백 (직설) | 2줄 |
| Final Chorus | Chorus + 감정 한 단계 상승 | 4줄 (AAB는 3줄) |
| Outro | Verse 1 장면 회귀 + 감정 변화 | 3~4줄 |

## 출력 포맷 규칙
- [Verse 1] 뒤에 아무것도 붙이지 않음 (감정%, 장면 설명 금지)
- 제목 뒤에 (라임구조) 괄호 금지
- A, B 라임 마커 금지
- 송폼 태그만 깔끔하게
