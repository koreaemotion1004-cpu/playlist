# Controlled Variation 가이드

## 원칙
Core DNA는 고정. 트랙별로 1~2개 축만 크게 바꾼다.
= 같은 세계관의 다른 트랙 (앨범 퀄리티)

## 변주 가능한 7가지 축
| 축 | 영향도 | 예시 |
|---|--------|------|
| BPM | 높음 | 62~82 범위 내 조절 |
| Groove/Feel | 높음 | Lazy → Lonely → Warm → Hazy |
| Arrangement | 높음 | Full trio → Piano solo → Bass+Piano only |
| Scene | 중간 | Empty café → Rainy window → 2AM → Closing time |
| Texture | 중간 | Light crackle → Heavy crackle → Smoky haze → Rain ambience |
| Vocal | 중간 | Whispered → Raw emotional → Sleepy → Warm smiling |
| Melody | 중간 | Mesmerizing → Melancholic → Cheerful → Nocturnal |
