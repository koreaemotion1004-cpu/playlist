# Noir 가사 공식 (Line 2, Vol.13~)

> **⭐ 가사 메타태그 마스터 참조: [[lyrics-meta-tags]]** (4개 라인 통합)
> Noir 권장: `[melancholy piano]` `[horns mournful]` `[muted trumpet, low]` `[detective cool]` `[reverb tail]` `[the last note of the night]`
> Noir 부정: `NOT bright` / `NOT happy` / `NOT pop production` / `NOT soaring`

## 핵심 원칙
제목 = 훅 = Chorus 첫 줄 (기존 빌보드 공식 유지)
테마/보컬/송폼만 Noir 전환. 악기 편성은 기존과 동일.

## 악기 편성 (기존 Core DNA 그대로)
```
strictly no horns, no trumpet, no saxophone, no brass
piano trio + upright bass + brushed drums + muted guitar
```
→ Noir 전형 악기(색소폰/트럼펫) 안 씀. Mysig 호환 스타일 유지.

## ⭐ 가사 안에 메타태그 삽입
섹션 헤더 바로 밑에 보컬 디렉션을 대괄호로 넣는다. Suno가 각 구간 보컬 톤을 다르게 잡음.

```
[Verse 1]
[low, intimate, almost spoken]

...

[Bridge]
[more breath, less projection]

...

[Outro]
[barely above a whisper]
Mmm, the room goes warm
...
```

## Bridge 규칙 (Noir 버전)
- 4줄 짧은 라인 (5~6음절 권장)
- 직설적 고백 or 반복 이미지
- 예시:
```
If you leave, leave slow
Like the last note goes
Like the vinyl hum
When the song is done
```

## Outro 규칙 (Noir 버전)
- "Mmm" 허밍 3~4줄
- 각 Mmm 뒤에 짧은 이미지 문구
- 감정 여운 + Verse 1 장면 잔상
- 예시:
```
Mmm, the room goes warm
Mmm, the night stays low
Mmm, don't say goodbye
Just let it go
```
