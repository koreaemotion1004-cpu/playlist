# HIT DNA 공식 — 7요소

## 출처
Suno 프롬프트 레이어 6단 구조 + 7가지 스위치 이론

## Suno Custom Mode 입력 필드
| 필드 | 역할 |
|------|------|
| Style of Music | 장르, 질감, 보컬, 장면 |
| Lyrics | 가사 내용 + 구조 힌트 + 플로우 |
| Advanced Options (Exclude) | 빼고 싶은 요소 지정 |

## 7요소 (강한 프롬프트 = structured prompt)
```
Genre:     장르/서브장르
Tempo:     BPM
Groove:    리듬 결, 그루브
Texture:   악기 종류보다 질감 표현
Vocal:     breathy / powerful / 보컬 에너지
Scene:     campfire / city / rain / 장면감
Structure: 흐름, 콜드 오픈, 후렴
```

## 약한 프롬프트 vs 강한 프롬프트
| 약한 | 강한 |
|------|------|
| sad acoustic song with guitar | Genre: warm indie acoustic folk / Tempo: 78 bpm / Groove: fingerstyle guitar rhythm / Texture: organic acoustic / Vocal: intimate storytelling / Scene: campfire night / Structure: slow emotional chorus |

## 프롬프트 패턴 라이브러리 (카페 재즈 관련)
- Rain Window: 비 오는 창가
- Late Night City: 도시 야경
- Lo-fi Chill: 공부/배경음

## 초보자 체크리스트
### 하지 말 것
- 악기만 바꾸고 BPM은 그대로 두기
- 감성 형용사만 잔뜩 늘리기
- 가사 길이/후렴 반복 구조를 그대로 두기
- 장면감 없이 장르만 적기
- Exclude / Personas / Inspire를 안 써보기

### 이렇게 바꾸기
- Core Prompt 1개를 먼저 고정하기
- Variation은 1~2개 축만 크게 바꾸기
- Lyrics box에 구조 / 장면 / 흐름 힌트 넣기
- 8~12개를 뽑고 선별하기
- 좋은 곡은 Personas / Inspire로 세계관 유지하기

## 앨범 워크플로우
```
Step 1: Core Prompt 고정 (장르, 보컬, 질감, 세계관)
Step 2: Variation 4~10개 (BPM / 리듬 / 구조를 크게 변경)
Step 3: Lyrics 보정 (가사 길이, 후렴 반복도 조정)
Step 4: 선별 & 보관 (좋은 곡은 Persona / Inspo 후보)
```

## 한 줄 결론
1개의 Core Prompt + 4~10개의 Controlled Variation = 플레이리스트 / 앨범 퀄리티
