# 네오소울 (Neo Soul) 공식 — Line 4

**Soul Funk에서 분리된 독립 장르.**
모던 소울·재즈 R&B 감성 곡 전용.

## 왜 분리했나
- **Soul Funk (Line 2):** 1970s 빈티지, Raw Dry Groove, Bill Withers 스타일, 남자 Baritone
- **Gospel (Line 3):** 남자 Tenor 고음, 찬양·성경
- **Neo Soul (Line 4):** 모던 프로덕션, Rhodes 중심, D'Angelo/Erykah Badu, 부드러운 보컬

제작·청취 상황·썸네일·보컬 방향 전부 다름.

## 핵심 정체성

| 요소 | 값 |
|------|---|
| 시대감 | 2000s~현대 (모던) |

## Soul Funk vs Neo Soul 차이점

| 요소 | Soul Funk (Line 2) | Neo Soul (Line 4) |
|------|-------------------|-------------------|
| 시대 | 1970s 빈티지 | 2000s~현대 |
| 베이스 | Upright, staccato | Fender, 깊은 groove |
| 드럼 | Ultra-dry, tambourine | Crisp, modern snare |
| 키보드 | Rhodes + Organ | **Rhodes 위주** |
| 보컬 | Baritone 남자 | Smooth 남녀 |
| 질감 | Raw, analog, warm | Polished + warm |
| 텐션 코드 | 적음 | **7, 9, 11th 많음** |
| 테마 | 철학·정체성·성경 | 일상·사랑·힐링 |

## ⭐ Suno Style of Music 최종 잠금본 (BPM 75 확정 · 2026-04-24 대수술)

**용도:** 정통 Neo Soul Funk (Mac Ayres / D'Angelo / Tom Misch 레퍼런스 정공법)
**보컬:** Deep Warm Male Baritone
**BPM:** 75 (네오 소울 특유의 느긋함. 70은 늘어지고 80은 과함)

### 🚨 2026-04-24 근본 수정 이유
기존 잠금본이 Cafe BGM 언어 잔존으로 **축 처지는 문제** 발생 (Soul Funk와 동일 패턴). A/B 테스트로 확인:
- ❌ `[Late-Night Cafe Mood]` → Cafe 느낌으로 변질

- ❌ `[Late-Night Cafe Mood]` → `[Late-Night Lounge Mood]`

### ⭐ 필수 고정 태그 4개 (스승님 공식 · 절대 변경·삭제 금지)
모든 Neo Soul Funk 프롬프트 **맨 앞에 반드시 포함**:
```
[Hypnotic Bass]
[Mesmerizing Melody]
[Neo Soul Funk]
[Mid-Tempo]
```
→ 이 4개는 **장르 정체성 DNA**. 어떤 변주·무드 조정에도 빼지 않는다.
→ `[Neo Soul Funk]` 태그는 Line 3 Soul Funk와 구분하기 위해 반드시 `Neo` 접두사 포함.
→ BPM 태그(`[75 BPM]`)는 이 4개와 **별개**로 바로 뒤에 배치.

## ⭐ 10곡 앨범 변주 배치 규칙 (기승전결 × 2변주)

**원칙:** 85점(그루비)과 95점(정돈)을 기승전결 곡선에 전략 배치.

### 배치 법칙
| Phase | 트랙 | 변주 집중 | 이유 |
|:---:|:---:|---|---|
| **起** (Opening) | T01~02 | **85점** (그루비) | 초대감·present vocal·song-like |
| **承** (Settling) | T03~05 | **85점 위주 + 95점 섞기** | 정착·리듬 타기·감정 안착 |
| **轉** (Depth) | T06~08 | **95점 위주 + 85점 섞기** | 깊이·BGM 침잠·무게 전환 |
| **結** (Closing) | T09~10 | **95점** (정돈) | 잔향·Steady Groove·마감 |

### 핵심 전환점
- **T05 → T06** 에서 85점 → 95점 완전 전환 = 기승전결 클라이맥스 전환점

### 권장 10곡 분포 예시
```
T01  🎸 85  (起 - 초대, 그루브 시작)
T02  🎸 85  (起 - 리듬 올라옴)
T03  🎸 85  (承 - 정착)
T04  🛋️ 95  (承 - 감정 안착 + 침잠 힌트)
T05  🎸 85  (承 - 그루브 주인공 · 마지막 상승)
T06  🛋️ 95  (轉 - 전환점, BGM 침잠 시작)
T07  🎸 85  (轉 - 마지막 리듬 강조)
T08  🛋️ 95  (轉 - 가장 정돈된 순간)
T09  🛋️ 95  (結 - 잔잔한 마무리)
T10  🛋️ 95  (結 - Steady Groove 여운)
```

**분포 요약:**
- 85점: 5곡 (T01, T02, T03, T05, T07) — 전반 집중
- 95점: 5곡 (T04, T06, T08, T09, T10) — 후반 집중

### Phase 내 번갈이 규칙
같은 변주 4곡 이상 연속 금지. 단, 結(T09~10)는 95점 연속 허용 (마감 일관성).

### 가사·변주 매칭 로직
- 초대·환영·리듬 가사 → 🎸 85점 (present, groove-forward)
- 내면·깊이·침잠 가사 → 🛋️ 95점 (restrained, minimal)
- 마감·여운 가사 → 🛋️ 95점 (steady, replayable)
