## 3. 바이럴 해부 — Vol.5 "No Need to Explain"

- **조회수**: 34,900회 (채널 최고)
- **좋아요**: 905개
- **좋아요율**: 2.6%
- **VPH**: 42.3
- **Outlier 배수**: 36.8x ⭐⭐⭐ (채널 평균 대비 36.8배)
- **길이**: 1:48:08 (채널 최장편)
- **성장 패턴**: 초기 완만 → D+10일경 급격 상승 (알고리즘 픽업 형태)
- **특이점**: 채널 전체에서 유일한 36x 이상 배수

### Vol.5가 바이럴된 이유 (가설)
1. **최장 길이 1:48** → 총 시청 시간(Watch Time) 최대치 → 알고리즘 가중
2. **부제 "No Need to Explain"** → 2인칭 철학적 어구 중 가장 범용적 공감대
3. **Vol.4(6x) → Vol.5(36.8x)** 패턴: Vol.4에서 알고리즘 임계점 돌파 후 Vol.5 폭발

## 4. 제목 공식

```
Soul Funk Groove | [2인칭/철학적 감성 어구] | DeepPocketStudio Session Vol.X
```

### 부제 패턴 카탈로그

| 유형 | 예시 |
|------|------|
| You + 불필요한 행동 부정 | You Don't Have to Say It / You Don't Have to Try |
| 존재/느낌 긍정 | You Already Know / You Feel It First |
| 무명·이름없음 | It Moves Without a Name / It Was Never Something You Had to Hold |
| 항상/언제나 | It Was Always There / Groove first. Always. |
| 직접적 선언 | No Need to Explain / No Rush Tonight / Slow and Steady |

**공통 특성:**
- **2인칭(You) 또는 3인칭(It)** 위주
- 설명하지 않아도 알 것 같은 **공감 어구**
- 명령형 X, 압박 X → **느긋하고 자연스러운 어감**
- 평균 5~7단어

## 7. 업로드 패턴

- **빈도**: 주 2~3회 (12편 / 약 4주 = 주 3회)
- **Vol.1~3**: 거의 동시 업로드로 채널 런칭
- **Vol.4~12**: 매 2~3일 간격 꾸준히 업로드
- 알고리즘 임계점 도달 전 포기 없이 지속 → Vol.5에서 폭발

## 8. 채널 알고리즘 임계점 분석

```
Vol.1~3: 평균 900회 (런칭 구간, 노출 최소)
Vol.4:   5,700회 ← 임계점 돌파 (6x 아웃라이어)
Vol.5:   34,900회 ← 폭발 (36.8x 아웃라이어)
Vol.6:   6,600회 ← 하강 (Vol.5 여파로 높은 기저)
Vol.7~:  1,200~2,000회 (정상화 구간)
Vol.12:  1,500회 + 36 VPH ← 최근 2일차, 상승 가능성
```

**패턴**: 신규 채널 기준 **Vol.4~5 구간(4~5주차)**에서 알고리즘 첫 픽업 발생.
우리 채널 BoomSlick 전략과 동일한 "꾸준 업로드 → 임계점 폭발" 구조.
