## 🎯 우리에게 적용할 인사이트 (골수까지)

### 1️⃣ ⭐⭐⭐ 제목 즉시 전환 (D0 발행 시 적용)
```
이전:  DEEP RELAX JAZZ ☕ Warm Cozy Swing | 1940s Cafe BGM 心ほどけるスロージャズ
                                          ↓ 전환 ↓
신규:  Quiet Morning Cafe Jazz | Relaxing 1940s Home Café Playlist | A Quiet Day
```
- Mysig 풀(VPH 11~57) → Velmora 풀(평균 VPH 105) **2~10배 효율**
- 일본어 빼고 영문만 (Velmora 검증) — 일본어가 오히려 트래픽 좁힘 가능

### 2️⃣ ⭐⭐⭐ 부제 단순화 — 단어 3~4개 형용사+명사
```
❌ 心ほどけるスロージャズ (3~4 단어 + 일본어 + 시적)
✅ A Quiet Day / Coffee Time Moments / Soft Light Hours
```

### 3️⃣ ⭐⭐ 가사 길이 확장 — 곡당 40~50줄
- 우리 신규 B 패턴 (26줄)도 부족
- Velmora 챔피언은 **추가 Chorus 반복 + Verse 더 길어**
- 다음 vol부터 **C 패턴 신설:** V1 → C → V2 → C → B → C → V3 → C → Outro = ~40줄

### 4️⃣ ⭐⭐⭐ 시그니처 훅 도입 (채널 정체성)
Velmora 시그니처:
- "Take it slow, let it flow"
- "Like a drifting melody"
- "Feel the day, let the quiet lead the way"

우리 시그니처 후보 (Cafe Jazz Line 1 신설):
```
"Pour it slow, let it stay"
"Like a quiet afternoon"
"Hold the cup, hold the day"
"Let the morning take its time"
```
→ 매 vol 1~2개 트랙에 반복 등장시켜 채널 정체성 형성

### 5️⃣ ⭐⭐ AABB 강한 라임 강제
```
이전 (vol.10~14): 약한 라임, 자유로운 흐름
신규 (vol.15~):  AABB 의무 — 매 절 첫 두 줄 강한 끝운
```

예시:
```
Tea leaves dancing in the cup     ← cup
Warmth is slowly rising up         ← up (강한 라임)
Morning light upon the table       ← table
Soft and stable                    ← stable (강한 라임)
```

### 6️⃣ ⭐⭐ 어휘 라이브러리 도입 (Velmora 검증 단어 90% 채택)
다음 vol부터 트랙당 다음 어휘 풀에서 70% 이상 사용:
```
빛: soft light, morning light, sunlight, golden, glow
움직임: drifting, dancing, flowing, gentle, breeze
속도: slow, quiet, calm, easy
사물: coffee, tea, cup, hand, window, curtain, table
시간: morning, day, moment, hour
```

### 7️⃣ ⭐ 트랙리스트 제거 (체류시간 최대화)
- 현재 우리 설명글에 트랙리스트 포함 → 점프 유도
- Velmora는 트랙리스트 없음 → 처음~끝 통째로 듣게 됨
- **다음 vol부터 트랙리스트 제거 또는 접기 처리** A/B 테스트

### 8️⃣ ⭐ 설명글 톤 친근화
- 시적 → 일상 친근
- "let every note warm your quiet room" → "soft sunlight, a gentle breeze, and a warm cup of coffee"
