# 가스펠 (Gospel) 공식 — Line 3

**Soul Funk에서 분리된 독립 장르.**
찬양·복음·성경 기반 강한 에너지 곡 전용.

## 왜 분리했나
- Soul Funk (Line 2) = **Baritone 기본 / 차분·그루비 중심**
- Gospel (Line 3) = **Tenor 고음 / 에너지 폭발 중심**
- 청취 상황 다름 (카페 배경음 vs 교회/찬양)
- 썸네일·제목·설명 전략 다름

## 핵심 정체성

| 요소 | 값 |
|------|---|
| 음역 | **Tenor / Falsetto (고음)** |
| 델리버리 | **Full-Chest Gospel** |
| 에너지 | 강한 힘·선포·열정 |
| 주제 | 성경·찬양·선포 |
| 악기 | Hammond B3 + Rhodes + Dry Drums + Tambourine |
| BPM | 90~110 |

- **BOOM PULSE** — 성경 기반 가사 완벽 구현
  - Ecclesiastes, Job, Esther, Exodus 등
  - 42만뷰 영상 보유

## 가사 공식

### 주제 공식 (한 곡 = 한 성경 장/구절)
- 전도서 1장 → Nothing New / Round and Round
- 욥기 1:21 → Naked I Came / Blessed Be the Name
- 에스더 4:14 → For Such a Time as This
- 출애굽기 → Prince of Egypt / Forty Years
- 데살로니가전서 5 → Don't Quench the Fire / Rejoice Always
- 시편 23 → The Lord is My Shepherd
- 시편 42 → As the Deer Pants
- 고린도전서 13 → Love Is Patient
- 갈라디아서 5 → Fruit of the Spirit

### 훅 공식
- 성경 핵심 구절 **원문 압축** (4~6음절)
- Chorus에서 3~5회 반복
- Call & Response 가스펠 패턴

### 대표 훅 예시
- "Blessed be the name"
- "For such a time as this"
- "Nothing new under the sun"
- "If I perish, I perish"
- "The Lord is my shepherd"
- "Don't quench the fire"

### 7단계 생성 프로세스
1. 성경 한 장 선정
2. 핵심 구절 3~5개 추출
3. 훅 선정 (4~6음절)
4. 현대 영어 각색
5. 소울펑크 어휘 매핑
6. Call & Response 삽입
7. 제목 결정

## 메타태그 금지
- `[intimate whisper]` ← 가스펠과 충돌
- `[laid-back baritone]` ← Soul Funk 쪽
- `[sultry]` ← Noir 쪽
- `[breathy]` ← 가스펠은 전성 낼 것

## 송폼 구조 (가스펠 버전)
```
[Intro] (Hammond swell + "Oh Lord" 애드립)
[Verse 1] (성경 장면 설정)
[Pre-Chorus] (텐션 빌드, 코러스 진입)
[Chorus] (핵심 훅 3~5회 반복 + Call & Response)
[Verse 2] (주제 심화)
[Pre-Chorus]
[Chorus]
[Bridge] (직설 선포 / 기도)
[Final Chorus with ad-libs] (훅 + 전성 + 고음)
[Outro] (Hey hey, Yeah Lord, 반복 훅)
```

## 가사 예시 (BOOM PULSE 분석 기반)

### 훅 반복 예시
```
[Chorus]
Blessed be the name
Blessed be the name
Naked I came, naked I go
Blessed be the name
```

### Call & Response 예시
```
[Bridge]
(Leader) "Don't quench the fire"
(Response) "No, no!"
(Leader) "Don't quench the fire"
(Response) "Let it burn!"
```

- Sepia Lounge 방식 차용 가능:
  - "For Such a Time. 🙏 Vintage Gospel Soul for Morning Prayer"
