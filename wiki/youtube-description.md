# YouTube 설명글 최적화 공식

## 구조 (5파트)
```
1. 첫 줄 (100자 이내) — 핵심 키워드를 자연스러운 문장에 삽입
2. CTA — Subscribe/Like (접히기 전 상단 배치)
3. 트랙리스트 — 타임스탬프 (각 트랙 사이 빈 줄 추가)
4. GOOD FOR 섹션 — 검색 의도 키워드 일본어로 삽입
5. 채널 소개 + 태그 — 키워드 한 번 더 반복
```

## SEO 포인트
- 첫 2~3줄이 핵심 — YouTube는 설명글 첫 줄을 검색 메타데이터로 읽음
- 제목 키워드 블록을 설명글 본문에 그대로 삽입 (교차 노출 강화)
- 작業用BGM, 睡眠用BGM, カフェBGM 같은 수요 키워드 삽입
- 설명글 충분히 길게 (알고리즘이 읽을 텍스트 확보)

## 해시태그 규칙
- YouTube 공식 권장: 3~5개
- 너무 많으면 스팸 처리될 수 있음
- 가장 강한 키워드를 앞 3개에 배치

## 타임스탬프 규칙
- 00:00부터 시작 (챕터 자동 생성 조건)
- 각 트랙 사이 빈 줄 추가 (유튜브에서 줄바꿈 유지)
- 마크다운 인식 안 됨 — 일반 텍스트로 작성

## 태그 라인 포맷
- # 제거, 콤마로 구분
- 예: スロージャズ, 作業用BGM, SlowMorningJazz, CafeJazz
