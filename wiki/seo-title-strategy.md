# SEO 교차 노출 제목 전략

## 핵심 원칙
조회수 잘 나온 기존 제목의 키워드 블록을 새 제목에 그대로 삽입하여,
시크릿 모드 검색 시 기존 영상과 새 영상이 동시 노출되게 한다.

## 전략 흐름
1. 기존 히트 제목에서 핵심 키워드 블록 추출
2. 새 제목 앞쪽에 새 감성/장면 키워드 배치
3. 뒤쪽에 기존 키워드 블록을 끊지 않고 삽입
4. 시크릿 검색 → 기존 + 새 영상 동시 노출

## 2개 제목 동시 노출 시
- A, B 두 제목의 공통 블록 찾기
- 공통 블록이 없으면 각각의 핵심 블록을 하나의 새 제목에 둘 다 넣기
- 너무 길면 A용 5개 + B용 5개로 나누기 (비추천 — 동시 노출 안 됨)

### 확정 공식
```
제목 = [자유 구성 부분] ｜ [히트 블록 원문 그대로]
         ↑                    ↑
         클릭률·차별화 담당      트래픽 상속 담당
```

### 적용 규칙
1. **히트 블록은 한 글자도 바꾸지 않는다** — 구두점, 공백, 대소문자 포함
2. **앞쪽은 자유** — 클릭률 최적화에만 집중
3. **구분자 ｜는 시각 효과용** — 매칭에는 영향 없음
4. **`Home Cafe Jazz` 같은 자체 브랜드 키워드**를 앞에 넣으면 기존 채널 트래픽까지 상속
