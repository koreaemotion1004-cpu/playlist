# 로맨스 소설형 설명글 공식

**Line 1 (Vintage Cafe Jazz) 전용 설명글 공식.**
Churipli 채널 나노 분석 + 로맨스 장르 결합.

## 왜 로맨스인가

우리 음악 (Vintage Japanese Jazz Kissa, 1940s~70s, 따뜻한 카페, Breathy 보컬)과
**로맨스 장르가 완벽 매칭**. 미스터리는 Noir 시리즈(Line 2)에 맡김.

## Churipli 구조 공식 (그대로 차용)

1. **첫 문장 = 감각 + 이질적 배경** (1인칭)
2. **전문 용어 1-2개** (재즈·커피·빈티지 지식)
3. **2~3명 등장** (주인공 + 이방인 + 조연)
4. **이국적 지명 + 실명 디테일** (실제 브랜드·좌표)
5. **짧고 깊은 대사**
6. **열린 결말 + 상징 이미지**
7. **길이: 2,500~3,000자**

## 로맨스용 수정 포인트

| 요소 | Churipli (미스터리) | 우리 (로맨스) |
|------|------------------|-------------|
| 핵심 긴장 | 미스터리 트릭 (19Hz, 아나모포시스) | 짧은 만남·재회·이별 |
| 결말 | 반전 / 공포 / 죽음 | 여운 / 약속 / 상징으로 남음 |
| 톤 | 차갑고 우울 | 따뜻하고 애틋 |
| 전문 용어 | 과학·건축·심리학 | **재즈 역사·커피·빈티지** |

## 인물 구조 (로맨스 공식)

### 1인칭 화자 유형
- 조율사, 작곡가 지망생, 전쟁 미망인, 번역가, 재즈 칼럼니스트, 사진가

### 이방인 남/여 유형
- 제대한 군인 + 음악가, 외국인 외교관, 옛 선생님의 제자, 피아니스트

### 조연 (공간 지킴이)
- 카페 주인, 바텐더, 호텔 벨보이, 음반가게 주인

## 스토리 비트 (한 곡 = 한 장면)

```
Beat 1: 비 오는 오후, 카페 입장 (감각 + 공간 소개)
Beat 2: 이방인 발견 (외형 디테일 + 레코드/책/담배 등 소품)
Beat 3: 첫 대화 (짧고 깊음)
Beat 4: 조연 등장 (공간 분위기 확장)
Beat 5: 숨겨진 사연 암시 (선생님/전쟁/이별)
Beat 6: 공유하는 순간 (음악/커피/침묵)
Beat 7: 작별 + 약속 (열린 결말)
Beat 8: 남겨진 상징 (레코드/편지/향기)
Beat 9: 화자의 여운 (마지막 한 줄)
```

## 전문 용어 사전 (재즈·카페 로맨스용)

### 재즈
- **78rpm, 45rpm, 33rpm** (레코드 속도)
- **Blue Note / Verve / Prestige / Impulse!** (레이블)
- **V-Disc** (WWII 군용 디스크)
- **C#4, A440, 평균율, 순정율** (조율)
- **Steinway, Bechstein, Bösendorfer** (피아노 브랜드)
- **브러시 드럼, 비밥, 쿨 재즈, 웨스트 코스트**

### 커피
- **Chemex, Melitta, French Press, Siphon** (추출 방식)
- **데미타스, 꼬르따도, 마키아토** (잔·음료)
- **93도, 블룸, 로스팅 프로파일** (기술 용어)

### 빈티지 디테일 (시대감)
- **Remington 타자기, Zenith 라디오, Victrola 축음기**
- **Chesterfield 담배, Old Fashioned, Manhattan** (음료)
- **트위드, 헤링본, 브로그** (패션)
- **Ford Coupe, Chevrolet Bel Air** (자동차)

## 대사 공식 (짧고 깊음)

### 형식
- 단문 (10~15자)
- 은유 + 사물 결합
- 답보다 여운

### 샘플
- "원래 모양이 그래요. 찌그러져 있어야만 보이는 것들도 있으니까"
- "커피는 혼자 마셔도 좋지만, 음악은 혼자 듣기 아까우니까요"
- "이 비가 그치면, 다시 만날 수 있을까요?"
- "음악은 기다리지 않아요. 사람만 기다리죠"
